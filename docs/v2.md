FreelanceFLow v2
Fonctionnalités à implémenter

1. Envoi de facture par email
L’utilisateur doit pouvoir :
envoyer une facture directement par email avec un message 
joindre automatiquement le PDF généré
2. Export des données
L’utilisateur doit pouvoir :
exporter la liste des clients en CSV
exporter la liste des factures en CSV
3. Dashboard enrichi
Vous devez enrichir votre dashboard avec les indicateurs suivants :
chiffre d’affaires total
nombre total de factures
chiffre d’affaires par client
Chiffre d’affaires par mois
Ces données doivent être calculées dynamiquement à partir des données existantes.
4. Personnalisation du PDF
L’utilisateur doit pouvoir :
uploader un logo
voir ce logo apparaître dans les factures PDF générées
5. Système de cache
Vous devez mettre en place un mécanisme de cache permettant :
d’optimiser certaines requêtes (ex: dashboard, listes)
de réduire les recalculs inutiles
Le choix de la stratégie est libre (mémoire, cache applicatif, etc.).
6. Logs structurés & monitoring
Vous devez mettre en place un système de monitoring simple permettant de visualiser :
l’état de l’application
les erreurs
éventuellement certaines métriques
Un dashboard de monitoring doit être accessible (outil libre).
Contraintes
Ces fonctionnalités doivent s’intégrer proprement dans votre architecture existante
Vous devez maintenir la qualité du code et des tests
Votre pipeline CI/CD doit continuer à fonctionner
L’application doit rester déployée et accessible
Gestion des backlogs propres
Release et plan de tag
