## Résumé exécutif

- **Clients :** l’adresse unique est remplacée par des champs structurés (`addressLine1`, `zipCode`, `city`, `country`), avec migration base de données associée (#45).
- **Authentification :** durée de vie des jetons ajustée, nettoyage des cookies obsolètes, signal `SESSION_EXPIRED` côté client (#44, #46).
